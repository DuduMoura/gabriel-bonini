Plano Técnico

1. Arquitetura da Solução

O sistema será desenvolvido utilizando uma arquitetura cliente-servidor baseada em aplicações web modernas.

Frontend: React
Backend (BaaS): Firebase
Banco de Dados: Firestore
Autenticação: Firebase Authentication

2. Tecnologias Utilizadas
Tecnologia	Finalidade

Firebase Authentication	Cadastro e login de usuários
Firebase Firestore	Armazenamento dos dados
Recharts	Geração de gráficos

Git	Controle de versão

6. Segurança
Autenticação por e-mail e senha utilizando Firebase Authentication.
Cada usuário terá acesso apenas aos próprios dados.
As regras de segurança do Firestore restringirão leitura e escrita aos usuários autenticados.
